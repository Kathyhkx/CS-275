Design:
 	The game has a 3*3 clickable grid associated with a piece of JavaScript control the 
 actions on the webpage.
 	The game will start with a empty grid and have players placing symbols in the grid 
 area. If either kind of symbol has 3 in a line, the certain player wins the game. If no
 one wins the game when all the fields are filled, this is a tie.
 
 Game Play Instruction:
 	Click on empty grid field to place symbol. Click on "Replay!" button to restart a 
 game.
